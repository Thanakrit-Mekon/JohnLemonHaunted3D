## Game explanation
This game use WASD or Arrow keys to control John Lemon to help him escape the haunted mansion and not get caught.
<br>There are two types of enemies in this game. They are static enemies and dynamic enemies. The static enemy is the collision object that stays in the same place. The dynamic enemy is the collision object that moves around. If John Lemon runs into the hitbox of the enemies, so he loses. Then the loose image transition will fade in.
